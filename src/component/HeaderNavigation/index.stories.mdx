# HeaderNavigation

PC에서 사용 가능한 헤더 네비게이션 컴포넌트 입니다. 헤더 컴포넌트에 추가하여 사용 가능합니다.

<ArgsTable of={SubNav} />


## Variations

### All (Default)

- 전체 서브메뉴가 한번에 노출되는 타입 입니다.


## 참고 사항

- 메뉴에 마우스오버를 할 경우, `is_active` 클래스를 추가하여 active 효과를 줍니다. (1,2뎁스 메뉴 동일)

- 현재 페이지에 해당하는 메뉴도 `is_active` 클래스로 active 효과를 줍니다. `NavItem` 컴포넌트에서 `is_active` prop으로 현재 메뉴 선택 여부를 제어할 수 있습니다. (1,2뎁스 메뉴 동일)
