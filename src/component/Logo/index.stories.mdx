# Logo

링크를 포함하고 있는 로고 컴포넌트입니다.<br/>
svg, png 등의 이미지 파일을 모듈로 추가하여 사용합니다.

### PNG 사용

<Canvas>
  <Story
    name="PNG"
  >
    {TemplatePng.bind({})}
  </Story>

컴포넌트 내에서 생성되는 `<a>` 태그에 `aria-label`이 제공되므로, `<img>` 의 `alt` 속성은 빈 값이어도 무방합니다.

```tsx
import IconReactLogoPng from 'asset/images/png/logo_react.png';

<Logo to="/" text="로고" logoWidth="36" logoHeight="36">
  <img src={IconReactLogoPng} alt=""/>
</Logo>

```

## 참고 사항

### HTML

`logoWidth`와 `logoHeight`은 CSS 변수로 선언되어 svg의 `width`, `height`를 지정하는데 활용됩니다.

```html
<a href="/" aria-label="로고" class="logo_xxx" style="--width:36px; --height:36px;">
  <svg>
  ...
  </svg>
</a>
```
