# Pagination

게시판 형식에 사용될 수 있는 페이지네이션 컴포넌트 입니다.

## Variations

### Default

- 최대 10개의 페이지 넘버를 제공하는 페이지네이션 입니다.
- 선택된 페이지를 제외한 나머지 페이지는 링크 이동이 가능합니다.
- 맨앞, 이전, 다음, 맨뒤 버튼을 제공합니다.

### 참고사항

- 페이지네이션 링크 클릭 시, `page` 파라미터로 페이지 인덱스가 추가된 url을 전달합니다.
- 현재 페이지 넘버가 변경될 때마다, 이전, 다음 버튼의 링크가 변경됩니다.
- 전체 페이지 넘버가 변경될 때마다, 맨뒤 버튼의 링크가 변경됩니다.
- 스토리북에서 controls의 currentPage 값 변경 시, 페이지네이션 링크의 변경된 href 값을 확인 할 수 있습니다.
